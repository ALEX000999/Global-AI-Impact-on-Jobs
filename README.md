# Global AI Impact on Jobs

## 📌 Project Overview
This project analyzes the impact of Artificial Intelligence (AI) on employment using statistical modeling and data analysis techniques.

## 📊 Dataset
- 300 observations (synthetic dataset)
- Countries: USA, Germany, China, UK, Canada, Russia
- Industries: Manufacturing, Healthcare, Finance, IT, Retail

## 🔍 Variables
- AI Adoption Rate
- Automation Risk
- Job Displacement
- New Job Creation
- Economic Impact Index

## ⚙️ Methods Used
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Sampling techniques (SRS, Stratified, Cluster)
- Probability analysis
- Simple Linear Regression
- Multiple Regression

## 📈 Key Findings
- AI adoption strongly increases job displacement
- Automation risk is a key mediator
- New job creation partially offsets job loss
- R² = 0.756 indicates strong model performance

- Comparative analysis chart
